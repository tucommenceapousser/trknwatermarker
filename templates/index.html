Ajoutez un Filigrane à Vos Images

by TRHACKNON

Bienvenue sur trknwatermarker, un outil en ligne qui vous permet d'ajouter un filigrane à une ou plusieurs images.

Modes disponibles :
- Si vous téléchargez une seule image, vous recevrez directement l'image avec le filigrane.
- Si vous téléchargez plusieurs images, celles-ci seront compressées dans un fichier ZIP.

Formats d'images acceptés : JPG, PNG, GIF, BMP
